## 🎯 About The Project

**El Fayoumi Cart** is a premium food delivery application that brings the authentic taste of Egyptian breakfast to your doorstep. This portfolio project showcases advanced Flutter development skills, implementing real-world e-commerce features with a culturally-rich design that honors Egyptian culinary traditions while providing a modern, seamless user experience.

### 🌟 Why El Fayoumi Cart?

- **🎨 Culturally-Rich Design**: Authentic Egyptian visual identity with modern UI/UX principles
- **🌍 Full Arabic RTL Support**: Complete Arabic support with Cairo font
- **⚡ Lightning Performance**: Optimized rendering with 60 FPS animations and instant load times
- **🛒 Smart Cart System**: Advanced state management with real-time price calculations
- **📱 Cross-Platform Excellence**: Pixel-perfect on both iOS and Android devices
- **🌙 Egyptian-Inspired Colors**: Green, Red, Yellow theme palette

## 🏗️ Project Architecture

```dart
// Models
class MenuItem {
  final String id;
  final String name;
  final String description;
  final double price;
  final String category;
  final String imageUrl;
  final bool isPopular;
  final double rating;
  final int reviews;
  final int preparationTime;
  final List<String> additions;
  final double? discount;
  final String size;
}

class CartItem {
  final MenuItem item;
  int quantity;
  List<String> selectedAdditions;
  double get totalPrice;
}

// State Provider
class CartProvider extends ChangeNotifier {
  List<CartItem> items;
  void addItem(MenuItem item, int quantity, List<String> additions);
  void removeItem(int index);
  void updateQuantity(int index, int quantity);
  void clearCart();
  double get totalPrice;
  int get itemCount;
}

// Main Screens
- HomeScreen        // Main screen with menu
- ItemDetailsSheet  // Product details and add-ons
- CartScreen       // Shopping cart and checkout
```
